# 参考资料

## 用途

存放调研过程中的外部资料、竞品分析、历史参考。

包括：

- 原版 QQ 纵横四海的核心机制分析
- 同类游戏（大航海时代、海商王等）的机制研究
- 技术调研（Prisma + SQLite、Next.js Server Actions 等）
- 相关领域论文或文章
- 行业数据或参考数值

## 重要性

参考文档是设计决策的基础。建议在每条参考资料中标注：

- 信息获取时间
- 来源链接（如适用）
- 与本项目的相关性

## 命名规范

```
source-topic.md
```

`source` 表明信息来源。

### 示例

```
qq-zongheng-sihai-core-mechanics.md
daikoukai-jidai-trade-analysis.md
prisma-sqlite-best-practices.md
```
