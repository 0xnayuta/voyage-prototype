# 待解决问题与讨论

## 用途

记录未解决的技术争议、待定设计、正在讨论的架构问题。

包括：

- 需要团队讨论的技术决策
- 尚未定案的设计方案
- 已知矛盾或冲突的约束
- 需要进一步调研的问题

## 生命周期

```
提出（创建文档）
  ↓
讨论（在文档中或通过沟通补充观点）
  ↓
决策（产生 ADR 或规格说明）
  ↓
归档（将问题文档移入 ../archive/）
```

## 命名规范

```
YYYY-MM-DD-short-title.md
```

日期为问题首次提出的日期。

### 示例

```
2026-06-25-price-volatility-algorithm.md
2026-06-26-ship-upgrade-curve.md
2026-07-01-event-system-scope.md
```
