# 项目定位

---

## 项目名称

纵横四海：单机版

## 项目类型

单人离线航海贸易经营游戏。

## 灵感来源

QQ家园《纵横四海》。

约束：

- 仅借鉴「航海贸易经营」核心玩法
- 不得复制原游戏文案、美术、角色、剧情、地图设计
- 不追求与原版一致的体验，只借鉴其核心循环

## 目标

现代化重构航海贸易经营体验——将原版 WAP 文字游戏的核心循环用现代 Web 技术呈现，保留回合制策略深度，去除联网社交负担。

## 核心循环

```
查看港口价格
  ↓
购买商品
  ↓
选择目的港
  ↓
航行（触发随机事件）
  ↓
抵达港口
  ↓
出售商品 → 获得利润
  ↓
升级船只
  ↓
继续贸易
```

## 非目标

- 不是复刻版
- 不是即时反应游戏
- 不是社交游戏
- 不是剧情驱动游戏
- 不是 MMORPG

## 目标平台

PC

## 目标用户画像

- 曾在 QQ 家园玩过纵横四海的老玩家，想重温跑商体验
- 喜欢大航海时代系列、海商王等贸易模拟游戏的玩家
- 喜欢文字界面 + 策略决策的玩家
- 不需要社交、只想安静玩一局的玩家
